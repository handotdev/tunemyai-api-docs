---
title: "Get All Finetuned Models"
description: "This endpoint returns a list of all of your finetuned models."
openapi: "GET /models"
---

<Info>
  If there's a certain field you want to filter by, ping [support@tunemy.ai](mailto:support@tunemy.ai).
</Info>

If you'd like to download a specific model, you can use the `url` for the model in the response.
